# OATutor Adaptation for Engineering Modules

OATutor is designed for math education. FRAMES modifies it for engineering learning:

## Adapted Features
- Step‑based reasoning visualization
- Scaffolded explanations
- Adaptive hints
- Error analysis

## Engineering Extensions
- Simulation tasks
- Troubleshooting flows
- Hardware diagrams
- Procedural walk‑throughs

## Integration
Used only for LMS sections that benefit from guided reasoning.
