# Student Onboarding LMS

Mobile-first progressive web app for student training.

## Features
- Module browsing
- Section rendering
- Time + scroll analytics
- Resume‑where‑you‑left‑off
- Assignments by Team Leads

## Architecture
Frontend: React (PWA)
Backend: Flask/FastAPI
Database: modules, module_sections, module_progress, analytics tables
