# System Overview

FRAMES / Ascent Basecamp is a unified ecosystem built on:
- One PostgreSQL database
- Three user-facing applications
- Embedded agent swarm (partitioned)
- Notion integration for content management
- Optional OATutor-derived adaptive learning components

## Applications
1. Student Onboarding LMS (mobile-first)
2. Team Lead Module Builder (admin workspace)
3. Researcher Platform (analytics + AI predictions)
